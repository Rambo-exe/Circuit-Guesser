  Circuit Guesser

A mobile puzzle game focused on logical reasoning and basic electrical circuit concepts.

  About the Project

Circuit Guesser is a mobile game designed to improve players' logical thinking skills through circuit-based puzzles. The player is presented with different circuit configurations and must determine the correct solution using reasoning.

  Features
Puzzle-based gameplay
Progressive difficulty system
Logic-focused challenges
Simple and intuitive UI

  Technologies
Unity (C#)
Mobile Development (Android)
Object-Oriented Programming (OOP)
